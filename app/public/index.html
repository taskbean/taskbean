Run bean track inside any project folder. The agent skill installs there, and new tasks show up here automatically.

Windows (PowerShell)
PS> cd C:\Users\you\my-project PS> bean track
WSL / Ubuntu
$ cd ~/my-project $ bean track

On WSL? Taskbean stores data at ~/.taskbean/taskbean.db, which resolves to a different path inside WSL than on Windows. To have this app see WSL projects, run bean track from PowerShell using the project's UNC path (for example \\wsl$\Ubuntu\home\you\my-project).
